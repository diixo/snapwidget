
# snapwidget


## How to start:
* Run FastAPI-server:
```
uvicorn main:app --reload
```

* Run HTTP-server:
```
python -m http.server 5500
```

Open **ai-client.html** in browser.

## Content:

* **client.html** with **widget.js**
* **ai-client.html** with **ai-widget.js**
